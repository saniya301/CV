PROJECTS

  • Online Auction Management System:
    The project is based on database management system. The system has been designed to be highly-scalable and capable of supporting large number of bidders in an active auction.
  • Supermarket Billing System:
    The project is based on file structures. The project deals with the automation of supermarket this software will help salespersons in managing the various types of records pertaining to his/her customer.
  • Dream wedding:
    It is a web based project which acts as a bridge between potential users and wedding planners. This project provides an easy way to plan their special events. It comforts the user to enjoy each and every function along with their family, friends and relatives.
  • Animal Identification Using Footprints (Final year project):
    The software consists of images footprints from animals of known identity are collected, trained and stored in the database. The software also uses techniques like binarization and segmentation. Using image processing techniques an algorithm was created to segment and extract the best possible representation of the footprint. It is a non-invasive technique.
